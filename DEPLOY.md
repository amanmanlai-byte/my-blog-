# 帽子云部署说明

## 项目信息
- 名称：216的摄影与动漫空间
- 框架：Hexo + Butterfly主题
- 输出目录：public/

## 构建命令
`ash
npm install
npm run build  # 等同于 hexo generate
`

## 注意事项
1. 需要Node.js 20+环境
2. 构建后生成public/文件夹
3. 主题：Butterfly v5.5.5-b1

## 本地测试
`ash
# 安装依赖
npm install

# 本地预览
npm run build
hexo server
`
